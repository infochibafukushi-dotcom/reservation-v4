# reservation-v4 管理画面フル強化 本番版

GASなし / Cloudflare Workers + D1 / 本番運用前提

## 管理画面 実装内容
1. 認証・遷移・ログアウト
2. ダッシュボード統計（予約件数・未対応・確認済・完了）
3. 予約カレンダー運行枠管理（週移動・通常/時間外・個別/日/時間帯ブロック・当日予約最短時間・予約済み反映）
4. 予約一覧モーダル・詳細モーダル・ステータス更新・非表示
5. ロゴ/ブランド/GitHub連携項目/ロゴWebP変換アップロード
6. 文言カスタマイズ
7. 管理パスワード変更
8. 当日予約設定
9. メニュー・料金マスタ管理
10. トースト、ローディング、リトライ通信、ブートストラップ/範囲データ分割取得、公開側キャッシュ削除

## 設置
1. worker.js を Cloudflare Worker に上書き
2. D1 Binding は DB
3. その他ファイルを GitHub Pages に上書き
4. 初期管理パスワード: 1234
